# オタクLOG — Anime & Manga Tracker

Track everything you've watched and read.

## Features
- Wish list, In Progress, and Completed sections
- Progress tracking for episodes/chapters
- Star ratings and favorite character for completed titles
- Syncs across devices with your account

## How to use
👉 [Open the app](https://YOUR-USERNAME.github.io/otakulog)

Sign in with GitHub or create an account with your email.
